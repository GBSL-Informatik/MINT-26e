# Königsbergerbrückenproblem

von Fabian L. & Leo M.

Das Königsbergerbrückenproblem ist ein berühmtes Problem in der Mathematik, welches behandelt, ob es möglich sei, einen Rundgang durch die historische Stadt Königsberg so zu planen, dass man jede der sieben Brücken **genau einmal** überquert.

![Königsberg um 1750](./images/Konigsberg_bridges.png)

**Leonhard Euler** ist unter anderem bekannt dafür, im 18. Jahrhundert das Problem mathematisch gelöst zu haben. Dabei begründete er die Basis für ein heutiges Gebiet der Mathematik, der "**Graphentheorie**". Das Königsbergbrückenproblem kann man aber natürlich auch für ähnlich aufgebaute Städte beziehen.

**Aufgabe:** Unterhalb finden Sie **fünf Varianten** einer Stadtplanausschnittes. Das einzige variierende in diesen fünf Stadtplänen ist die Anzahl von Brücken. In jeder Variante hat es eine andere Anzahl von Brücken. Findet heraus, bei welchem der fünf Plänen man theoretisch einen Spaziergang machen könnte, bei welchem man über alle Brücken läuft, ohne eine Brücke zweimal zu überqueren. Wähle unterhalb die richtige Antwort A, B, C, D oder E aus.
