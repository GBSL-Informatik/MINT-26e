# QR Code Generator

QR Codes sind praktisch um Inhalte wie URLs oder anderes schnell zu teilen. 

```mdx
import QrGenerator from '@tdev-components/shared/QR-Code/Generator';

<QrGenerator text="Hello World" />
```

<BrowserWindow>
    <Generator text="Hello World" />
</BrowserWindow>

## Inhalt anzeigen

```mdx
<QrGenerator showText text="Hello World" />
```

    <Generator showText text="Hello World" />
</BrowserWindow>
