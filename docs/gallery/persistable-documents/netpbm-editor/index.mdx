# Portable Anymap Editor
Netpbm ist eine Familie von einfachen Dateiformaten zur Speicherung von Rastergrafiken.

Diese Komponente bietet einen Live Viewer und einen speicherbaren Editor für Netpbm-Bilddateien.

:::warning[Standardtreue]
Die Implementation ist nah am offiziellen Netpbm-Standard gehalten, kann aber gewisse Freiheiten in Anspruch nehmen, um die Benutzererfahrung zu verbessern.
:::

```mdx
<NetpbmEditor />
```

<BrowserWindow>
    <NetpbmEditor />
</BrowserWindow>

Mit dem Parameter `default` kann ein Standardwert für den Editor festgelegt werden, welcher beim ersten Laden des Editors direkt im Viewer angezeigt wird.
